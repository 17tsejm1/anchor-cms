Database

Your database settings are found in the config.php there are required to run Anchor:

'database' => array(
	'host' => 'localhost',
	'username' => 'myusername',
	'password' => 'mypa33wr0d',
	'name' => 'anchorcms'
)

For most hosting providers the database host will most likely be localhost.
