Requirements

When deploying Anchor your server will need the following:

  • Apache web server (with mod_rewrite enabled for clean urls).
  • PHP 5.3
  • MySQL 5

Installation

A few basic steps to follow:

  1. Download Anchor
  2. Extract the archive and upload the contents to your web server.
  3. Navigate to your application in a web browser and follow the install guide.
