Theming Anchor

Theming Anchor CMS aims to be easy.

The files

A theme for Anchor requires two files:

  • index.php
  • sub.php
  • about.txt
Note: the search.php file is also required if your site uses a search feature.

There are a number of optional files you can also include:

  • includes/header.php
  • includes/footer.php
  • posts.php
  • 404.php

So what do all of these files do? As the building blocks of your Anchor site, they do quite a lot.

about.txt required
The metadata for your theme. It has its own simple syntax, and is required to show as an available theme in the admin panel.
index.php required
The source for the homepage/index page of Anchor. Unlike Wordpress, this is not the fallback page (sub.php is).
sub.php required
The default page for subpages, and the fallback file for non-included files: if you don't have a posts.php file, for example, it will try to load the sub.php file instead.
