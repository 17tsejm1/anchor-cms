Files

Requires files to setup a theme:

  • about.txt
  • index.php
  • article.php
  • page.php
  • posts.php
  • search.php
  • 404.php

You can also provide a functions.php file to alter the output and format your data.

Global Functions

Inside each theme template you can use a php function to retrieve dynamic data. Here is a list of all the available defined functions.

Posts

has_posts()
Returns true if there is published posts or false when there is none.
posts()
Returns true when there is more posts to be listed or false when we are at the end. Recommened to only be used in while loops.
